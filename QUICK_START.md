# Быстрый старт платформы "Вундеркинд"

## 📍 Проект находится на рабочем столе в папке: `Wunderkind_Platform`

## 🚀 Вариант 1: Простой запуск (без бэкенда)

1. **Откройте папку проекта:**
   - Перейдите на рабочий стол
   - Найдите папку `Wunderkind_Platform`
   - Откройте ее

2. **Запустите сайт:**
   - Перейдите в папку `frontend`
   - Дважды кликните на файл `index.html`
   - Сайт откроется в вашем браузере

3. **Изучите функционал:**
   - Нажмите "Войти" в правом верхнем углу
   - Используйте тестовые учетные записи
   - Изучите главную страницу, курсы, расписание

## 🚀 Вариант 2: Запуск с бэкендом (Node.js)

### Шаг 1: Установите Node.js
1. Скачайте с официального сайта: https://nodejs.org/
2. Установите версию LTS (рекомендуется)

### Шаг 2: Запустите бэкенд
1. Откройте командную строку (CMD)
2. Перейдите в папку бэкенда:
```cmd
cd C:\Users\Ольга\Desktop\Wunderkind_Platform\backend
```

3. Установите зависимости:
```cmd
npm install
```

4. Запустите сервер:
```cmd
npm start
```

5. Сервер запустится на http://localhost:3000

### Шаг 3: Откройте фронтенд
1. Откройте файл `frontend/index.html` в браузере
2. Теперь сайт будет работать с API

## 🚨 Устранение неполадок:

### Проблема: Сайт не открывается
Решение: Убедитесь, что файл `index.html` открывается в браузере (Chrome, Firefox, Edge)

### Проблема: Бэкенд не запускается
Решение:
1. Проверьте, что Node.js установлен: `node --version`
2. Переустановите зависимости: `npm install`
3. Проверьте порт 3000: `netstat -an | find "3000"`

### Проблема: API не отвечает
Решение: Убедитесь, что бэкенд запущен на http://localhost:3000
